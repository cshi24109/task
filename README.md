# task

## アプリ概要
時間割・予定入力
時間割の入力画面では遅刻や欠席のカウントが行える.新学期や過去の時間割も作成できるようになっている.
カレンダーに変更も可能.予定が入力出来て，課題、テスト、その他の予定を入力できるようになっている.
